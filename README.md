This repository is reserved for **Finetune Workbench**, the **L2 training workbench** in the DOORM WorldModel-OS architecture — a model-agnostic LoRA fine-tuning pipeline with three hard quality gates.

Code will be released here under **AGPL-3.0-or-later** once the three release gates in §11 of the companion paper are passed.

**Base-model agnostic**: any decoder-class LLM that respects the data-format and `adapter_version` handshake can serve as the base. v0 reference: `Llama-3.2-3B-Instruct` (substitutable, not binding).
